# Plataforma de Aulas (Video Platform)

Um sistema full-stack focado no gerenciamento e hospedagem de conteúdo educacional em vídeo.
O projeto permite o upload de vídeos e a organização de links,
fornecendo uma interface de consumo rápida e eficiente para os alunos.

## 🛠 Tecnologias Utilizadas

**Front-End:**
* Next.js
* React
* Tailwind CSS (v3)

**Back-End:**
* Node.js
* Fastify
* PostgreSQL

## ⚙️ Funcionalidades

* Gerenciamento de uploads de vídeos.
* Organização de aulas através de links estruturados.
* Painel de administração (Dashboard) para controle de conteúdo.
* Consumo de API rápida e assíncrona utilizando Fastify.
